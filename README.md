# Sales Dashboard

This project demonstrates the use of Google Data Studio to build a simple interactive sales dashboard from transactional retail data. The dashboard summarizes key sales metrics and provides an overview of product performance, sales trends, and geographic distribution.

## Tools Used

- Python (Data Cleaning)
- Google Sheets
- Google Data Studio

## Dataset

- Dataset sourced from the [UCI Online Retail Dataset](https://archive.ics.uci.edu/dataset/352/online+retail)
- Cleaned using Python in [online-retail-data-cleaning-with-python](https://github.com/Raav-Paul/online-retail-data-cleaning-with-python)
- UK-based online retail transactions from December 2010 to December 2011
- 349,203 cleaned transaction records

The dashboard includes:

- KPI cards for Units Sold, Total Sales, Average Order Value, and Average Order Size
- Top 5 products by revenue
- Sales trend over the reporting period
- Geographic distribution of sales
- Interactive filters for date range and country

## Purpose

The objective of this project is to demonstrate the fundamentals of dashboard design using Google Data Studio, including data visualization, KPI reporting, interactive filtering, and basic business reporting.
