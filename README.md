# Sales Dashboard
This project demonstrates basic dashboarding using Data Studio. The goal is to transform a raw transactional dataset into a structured analysis dashboard.

# Tools Used

- Google Sheets

- Data Studio


# Dataset

- Dataset sourced from the [UCI Online Retail Dataset](https://archive.ics.uci.edu/dataset/352/online+retail)

- Data was cleaned in [online-retail-data-cleaning-with-python
Public](https://github.com/Raav-Paul/online-retail-data-cleaning-with-python)

- Contains transactional sales data from a UK-based online retailer between 2010 and 2011

- 349,203 transaction records across 8 columns

# Purpose

This repository demonstrates the use of Data Studio to make a sales dashboard.
